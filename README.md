# 📡 FFT Signal Analysis (MATLAB)

## 📌 Überblick
Dieses Projekt untersucht die Frequenzanteile eines Signals mithilfe der Fast Fourier Transformation (FFT).

---

## 🎯 Ziele
- Erzeugung eines Signals  
- Anwendung der FFT  
- Analyse des Frequenzspektrums  
- Visualisierung des Single‑Sided Spectrum  

---

## 🧠 Methoden & Tools
- MATLAB  
- FFT  
- Nyquist‑Theorem  

---

## 📊 Visualisierungen
- Zeitbereich  
- Frequenzbereich  
- Single‑Sided Spectrum  

---

## 📁 Projektstruktur
data/
notebooks/
src/
visuals/
docs/

---

## 🚀 Ergebnisse
Die FFT zeigt dominante Frequenzen klar und ermöglicht eine präzise Analyse des Signals.
